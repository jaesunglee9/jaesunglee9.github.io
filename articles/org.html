This document serves the purpose of learning about org-mode and org-roam.

What is org-mode?

An org file is a text file in org format, similar to markdown. Using org-mode, one can create documents of desired style and format.

What is org-roam?

Org-roam is a personal knowledge manager inspired by Roam software. It is built on top of org-mode.

How to use org-mode and org-roam

org documents serve as general Project documentation. How to generate Project code documentation should be considered later. Projects should have org files that are symlinked to ~/org. this allows org agenda to find all necessary org files without looking through all project directories. TODOs are captured to the project specific org files. This allows TODOs to migrate with the Projects but allow them to be seen by org-agenda. although the said files are symlinked( or synced with cron), This means that org directory, should not be exposed to the public. It will contain org files that belong in personal/private. also, this suggests that relevant services could be created, along with org setups.
