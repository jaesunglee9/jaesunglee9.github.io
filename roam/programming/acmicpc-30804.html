문제

Eunha made a Tanghulu (a Chinese snack made from sugar-coated fruits) with \(N\) fruits on a long stick. Each fruit is of a type numbered from \(1\) to \(9\), and starting from the front, fruits of types

\(S_1,S_2,\cdots ,S_N\) are pierced in order. Upon rechecking the order after making the fruit Tanghulu, she realized that she forgot the request to use only a maximum of two types of fruits.

Eunha, with no time to make another Tanghulu, decided to remove a number of fruits from the ends of the stick, leaving only a maximum of two types. After removing \(a\) fruits from the front and \(b\) fruits from the back, there are now \(N-(a+b)\) fruits of types \(S_{a+1},S_{a+2},\cdots ,S_{N-b-1},S_{N-b}\) on the tanghulu. \((0\le a,b;\)

\(a+b

Find the maximum number of fruits on the Tanghulu with a maximum of two types of fruits made using this method.

입력

The first line of input contains a single integer \(N\), denoting the number of fruits.

\((1\le N\le 200\, 000)\) 

The second line of input contains \(N\) space-separated integers \(S_1,\cdots ,S_N\).

\((1\le S_i\le 9)\) 

출력

The first line of output should contain the maximum number of fruits on a Tanghulu with a maximum of two types of fruits as described in the problem.

Approach

At any possible state of tanghulu(subsequence), it can be represented with two indices pointing to the start and end of the subsequence. This observation leads naturally to the two-pointer approach. two-pointer approach iterates over the sequence once, hence is O(N). It is probably one of the optimal solutions.

I initially tried greedy approach. It does not cover all possible states, and so is not correct.
