Linked List(Data Structure)

A linked list is a linear collection of data elements. These data elements, also called nodes, are not exclusive to linked lists, but are called so in trees, or any other structure where we wish to store any other value or pointer with said data. Linked list is a data structure which can in turn be used to implement other data structures, such as Queue(Data Structure), or Queue(Data Structure), and so on.

Linked Lists versus Arrays

Both are linear, but LLs are not stored in consecutive memory locations. Insertions and deletions can be done at any point in the list in constant time. Size of LL is variable during runtime, So LLs are good when many insertion and deletion needs to be done, and memory is not an issue.

Singly linked list

Every node contains two data-a value, and a pointer to the next node. If Node is last node, than the pointer stores a null value. Allows only one-way traversal

Circular linked list.

No null pointer at the end

make new node ptr to last node new node next to start ptr->next = newnode start = newnode

Insert at end
