Floyd-Warshall Algorithm

Function

  • Finds shortest paths between all edges.
  • Can handle negative edge weights.
  • Does not work with graphs with negative cycles.

Proof of Correctness

Time Complexity

O(V3)

Space Complexity

When to be preferred?

When the graph is dense. When it is sparse, Johnson's Algorithm is to be preferred.
