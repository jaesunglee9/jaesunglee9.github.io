So, To build a knowledge base, we need to define componenets of the world. Wall location Wall0,0 … (N symbols for N locations) Percepts BlockedW0 (Blocked to the west of agent at timestep 0) (4T symbols for T time steps) Actions W0 (Move west at time step 0) (4T symbols ) Location At0,00 (is at 0,0 at time step 0) (NT symbols)

O(NT) variables, and each can have binary values, so O(2NT) possible worlds

Now how to build knowledge base? Wall0,0 and Wall0,1 and Wall0,2 and ….. notWall1,1 and notWall1,2, and ….

The agent knows the above, but doesn't know it's location so its location is At1,10 or At1,20 or ….

How would the agent localize its location? With percepts

percept variable at t iff some condition on world at t

the percept limits possible worlds

Satisfiability

A sentence is satisfiable if it is true in at least one world

How to test entailment? alpha entails beta iff alpha implies beta in all worlds and not (alpha implies beta) is false so alpha and not beta is false in all worlds so assume negative of assumption, if it is satisfiable, than it is not entailed
