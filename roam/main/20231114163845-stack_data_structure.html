Stack(Data Structure)

Think of a napkin dispenser with spring in the bottom. When we fill it, we push each napkin down, and the last napkin to go in comes out first. The last-in first-out (LIFO) data structure Needed for function calls

Push

Adds element to the top of the stack.

Overflow

If TOP = MAX - 1, stack is full, and no element can be inserted

Pop

Removes element at the top of the stack.

Underflow

If TOP = NULL, the stack is empty, and no element can be deleted.

Peek

Returns value of the topmost element

Linked Representation of Stacks(Linked Stack)

If stack is small, or maximum size is known in advance, array implementation gives efficient implementation. However, if array size is indeterminiate i.e. if data size is variable or number of data is unknown, linked representation is used.

Operations

All insertions and deletions are done at the node pointed by TOP If TOP = NULL, then the stack is empty.
