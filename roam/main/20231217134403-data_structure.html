Data Structure

Data structures are group of data elements that are put together under one name, and which defines a particular way of storing and organizing data in a computer so that it can be used effectively.

Types of Data Structure

Array(Data Structure) Linked List(Data Structure) Queue(Data Structure) Stack(Data Structure) Binary Tree(Data Structure) Hash Table(Data Structure) Heap(Data Structure) Graph(Data Structure)

  • Searching
  • Inserting
  • Deleting
  • Sorting
  • Merging

Abstract Data Type

An abstract data type is a way of looking at data structures, essentially stating that we don't need to look at inner workings of a data structure and care about getting the data the user needs. Such is called encapsulation.
