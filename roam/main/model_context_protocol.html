What is a tool?

A function or program which an LLM-based application can use to accomplish a task ouside the model's capabilities. The model itself generates content, tools allow the application to interact. They allow a model to know something, or to do something.

Do

Call an External API, exectue some other code or function.

Know

Access structured and unstructured data sources
