Search Problem

*Problem solving agents There are two type of agents: Problem solving agents, and Planning agents. Problem solving agents use atomic representations, while planning agents use factored representations.

Problem solving agents solve the search problem.

Constituents

A search problem consists of: state space successor function start state goal state(s)

The above is the most general formulation, and the successor function may include action/result function, and also the cost function.

A search problem is solved using a search algorithm.

Searching with search tree expand potential plans(fringe) with node to explore?

Depth-First Search

Fringe is LIFO queue. Why? because We have to expand last visited node. Since it always searches to the edge node, time complexity is O(bm). space of the fringe is only the edge node, which is O(bm) not complete if infinite, since even if finite solution exists, it may not be able to find it if the tree is infinite for some node

Not optimal(not the shortest path)

Breadth-First Search

Fringe is FIFO stack. Why? because we have to expand the shallowest unexpanded node Since we search for the shallowest node, the fringe is no bigger than the size of the bs at the solution depth s complete optimal if costs are all 1

BFS vs DFS

BFS obviously optimal, and fast DFS memory is lower(space advantage)

Iterative Deepening

DFS with depth limit 1, 2, and so on

search space is larger than memory and depth of solution is unknown

Cost-sensitive search

Uniform-Cost Search

simple : when choosing which node to expand, use path cost heuristic

time complexity is O(b(1 +floor(C*/e))) complete(minimum arc cost is positive) optimal

Con: Explores options in every direction No information about goal direction
