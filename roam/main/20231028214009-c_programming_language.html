C Programming Language

Source code

  • Uses .c extension. This file contains C source code that defines main function among others.
  • main() is starting point of program execution.

Header files

  • Uses .h extension. This file contains function declarations and other things.
  • One might wonder why c/c++ has this unique files, when other PLs don't. Seems like this is due to C programs being linked to binary, in which you have to inform the compiler of the available functions.

Input/Output functions

  • The most basic is scanf/printf functions
  • printf takes the desired string format, and variables needed.
  • scanf takes expected string format, and the address to store the variables. Note. scanf can also take in strings, in which case there is no need for reference &
