Multicore Programming

Concurrency means within same time, Some tasks are executed simultaneously. On single cores, threads are interleaved. On multiple cores, threads are executed at the same time using multiple cores. The multi core concurrency is called Parallelism.

Challenges in multicore programming is:

  1. identifying tasks,
  2. balance
  3. data splitting
  4. data dependency
  5. testing and debugging.
