Fields(Mathematics)

A field is a fundamental algebraic structure that combines two main operations, addition and multiplication, and satisfies set of specific properties.

Properties

Closure

A field is closed under addition and multiplication, implying that for elements a, b in field F, a + b and a * b is also in field F.

Identity element

for All a, there exists an identity element eadd such that a + e = e + a = a. Namely, zero for All a, there exists an identity element emul such that a * e = e * a = a. Namely, one(or unity)

Inverse

for every nonzero element a in F has an additive inverse b such that a + b = b + a = 0 for every nonzero element a in F has an multiplicative inverse b such that a * b = b * a = 1

Associativity

a + (b + c) = (a+b)+c. Likewise, a*(b*c) = (a*b)*c Note: The above 4 properties makes a group

Commutativity

a + b = b + a ab = ba

Note: Satisfaction of commutativity along with the above first four, makes an commutative(abelian) group.

Distributivity

(a + b)c = ac + bc Note: The above properties are seemingly the same and independent of each operation +, *. To relate the two operations, we require property of distributivity, which defines the relationship between addition and multiplication.

Examples

N: Set of natural numbers. It is closed under addition and multiplication, but has no inverse for both addition and multiplication. Z: Set of integers. closed under add and mul, but, has no inverse for multiplication Q: Set of rational numbers. Is a field. R: Set of real numbers. C: Set of complex numers.
