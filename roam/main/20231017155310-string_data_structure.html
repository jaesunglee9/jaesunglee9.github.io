This is a bit of a misnomer imo because you are not extracting in the sense that you remove the substring from the main string; It leaves the main string intact.

Requires: main string, position of first character in the string length of substring

Step 1: INITIALIZE Set I=M, J=0 (set index to the first letter position substring, and j) Step 2: while str[I] != NULL and N > 0 (while not end of the string and chars left to copy) Step 3: set substr[J] = str[I] Step 4: set I = I + 1 (main string index +1) Step 5: set J = J + 1 (sub string index + 1) Step 6: set N = N - 1 (chars to extract -1) Step 7: set substr[J] = NULL Step 8: exit
