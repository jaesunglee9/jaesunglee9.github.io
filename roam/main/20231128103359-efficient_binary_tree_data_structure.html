Efficient Binary Tree(Data Structure)

A binary search tree, also known as ordered binary tree, is a variant of binary trees in which the nodes are aranged in an order. Specifically, It is arranged so that all nodes in left sub tree hvae value less than the root. Also right nodes have value greater than the root. Search takes O(log2 n) time. It also speeds up insertion and deletion operations, so has a speed advantage when data in the structure changes rapidly.
