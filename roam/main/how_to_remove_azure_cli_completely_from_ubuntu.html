How to remove azure cli completely from ubuntu

Like this:

```bash sudo apt-get remove -y azure-cli

sudo rm -f /etc/apt/sources.list.d/azure-cli.sources

sudo rm -f /etc/apt/keyrings/microsoft.gpg

sudo apt-get autoremove -y

rm -rf ~/.azure ```
