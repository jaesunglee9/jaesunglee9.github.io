Tree(Data Structure)

Tree is a hierarchical data structure, meaning that data are separated into different levels. Contrast this with arrays, where every data is in the same level. A tree is acyclic directed graph? This is the etymology i believe. Trees are recursive, meaning that a tree is composed of trees(subtrees). A tree has a root, a node can have only one parent.

Root node

topmost node. If R = NULL, the tree is empty.

Subtree

a node and its descendents constitute a tree. If the said node is not a root node, then the tree is a subtree.

Leaf node

a node that has no chidren. Also called a terminal node.

Ancestor node(s)

Set of all predecessor nodes on a given node

Descendant node(s)

Set of all successor nodes on any given node.

Path

A sequence of consecutive edges

Level

Every node is assigned a level number. The root node is level 0, while its children are incremented by 1.

Degree

The number of children a node has.

Complete binary tree

is a binary tree that satisfies two properties: every level is completely filled, all nodes are as far left as possible.

left child formula

Given index k of node, 2k is ndex of left child.

right child formula

2k + 1

parent formula

k/2

height

tree having n nodes is given as ceiling(log2(n+1))

Building binary tree from general tree

binary tree left child is general tree left child binary tree right child is general tree right sibling

Traversing binary tree

Pre order

visit root node traverse left subtree traverse right subtree

inorder

traverse left subtree visit root traverse right subtree

Post order

traverse left subtree traverse right subtree visit root

Level order traversal

breadth first traversal

Huffman Tree

binary tree of nodes Using priority queue, merge two lowest priority node and put them on priority queue with sum of priority. Continue until complete, use it to construct data coding
