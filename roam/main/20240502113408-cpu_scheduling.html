CPU Scheduling

Based on the fundamental idea that a process is comprised of CPU and I/O burst cycles, so applying CPU scheduling can maximize CPU usage

CPU scheduling takes place under the following four circumstances

  1. When a process switches from the running state to the waiting state(I/O or wait())
  2. When a process switches from the running state to the ready state. (Interrupt)
  3. When a process switches from the waiting state to the ready state(I/O completion)
  4. When a process terminates
