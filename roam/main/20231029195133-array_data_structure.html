Array(Data Structure)

Arrays are collection of similar data elements Generally of same data type. stored in consecutive memory locations, referenced by index

Declaration

Declaraing an array means you are specifying data type, name, size. This is intuitive in that what you want to do is to designate select block in computer memory so that it is addressable. You can't do that without knowing data type, or by not giving it a name, or how much data you will put in. type name[size];

Access

index starts from 0 to size-1 To traversere array, use loops

addresses

Addresses are consecutive, so A[k] = BA(A) + w(k - lower bound)

length

Length = upperbound - lowerbound + 1

Storage

  1. Initialization during declaration type arrayname[size]={list of values};
  2. Input elements from keyboard
  3. Assign values to individual elements

Operations

Traversal

Insertion

insertion at the end Step 1: Set upperbound = upperbound + 1 Step 2: Set A[upperbound] = VAL Step 3: EXIT Insertion in the middle Step 1: [INITIALIZATION] SET I = N - 1 (index of last array element) Step 2: REPEAT while I >= POS (shift right part of the array to the right) Step 3: SET A[I+1] = A[I] Step 4: SET I = I-1 [END OF LOOP] Step 5: SET N = N+1 (Increment array size) Step 6: SET A[POS] = VAL (Insert array element)

Search

Deletion

Deletion at the end of the array Step 1: SET upperbound = upperbound -1 Step 2: EXIT

Deletion from middle of the array Step 1: SET I = POS Step 2:REPEAT while I < N - 1 Step 3: SET A[I] = A[I+1] Step 4: SET I = I + 1 [END OF LOOP] Step 5: SET N = N - 1 Step 6: EXIT

Pointers and Arrays

concept of array necessiates pointers. why? for because memories must be accessed in contiguous order *ptr = arr; means that the pointer is pointing at arr[0] remember that increment operators take precedence before dereference operators

Arrays of pointers

int *ptr[10];

Two-dimensional arrays

arr[i][j] i: row, j: column

datatype arrayname[rowsize][columnsize]

m by n array: m * n data elements, i < m, j < n

Memory representation

Row major order

(0,0), (0,1), (0,2), … Address(A[i][j] = BA + w*N*i + w*j

Column major order

(0,0), (1,0), (2,0) … Address(A[i][j] = BA + w*M*j + w*i
