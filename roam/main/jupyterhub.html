JupyterHub

How to install in ubuntu

sudo apt update sudo apt install -y nodejs npm

sudo npm install -g configurable-http-proxy

sudo python3 -m pip install jupyterhub jupyterlab notebook

Generate config

sudo mkdir -p /etc/jupyterhub cd /etc/jupyterhub sudo jupyterhub –generate-config

Config

c.Authenticator.allowedusers = {'yourusername', 'teammate1', 'teammate2'}

c.Authenticator.adminusers = {'yourusername'}

c.JupyterHub.ip = '0.0.0.0'

c.JupyterHub.port = 8000

c.Spawner.defaulturl = '/lab'

run

sudo jupyterhub -f /etc/jupyterhub/jupyterhubconfig.py
