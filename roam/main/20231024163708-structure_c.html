Structure(C)

Refers to data structures in c

Definition

Structures can store related information even of different data types together

struct structname { int sfsf; char name[20]; }

after you defined it, you can initialize it as struct student stud1={01, "rahul", …}

Note that you can initialize it during definition, in which case the syntax is mildly different.

Typedef

the keyword typedef can come before struct during struct definition, in which case the struct becomes its own type.

partial initialization

During initialization of a struct, you do not need to give entire data to the struct, in which it is called partial initialization. The uninitialized values are assigned default values. I note that there doesn't seem any way to specify which variable to assign.

accessing members

just use dot operator structvarname.membername

of course, using & in front of the members makes it the address.

copying structures

declare one struct and leave another uninitialized struct student stud1={blah blah}; struct student stud2;

stud2=stud1;

However, you can't compare it like stud2 == stud1. you have to do it yourself.

Nested structures

A structure that contains another structure

Arrays of structures

struct student stud[30];
